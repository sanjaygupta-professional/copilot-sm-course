## Sprint Review Visuals

Sprint reviews benefit from professional visuals that engage stakeholders.

### Exercise 1: Sprint Review Slide Background

```
Create a professional background image for a Sprint 47 Review presentation.

Theme: Technology and progress
Style: Modern, corporate, clean
Include: Abstract tech elements (lines, nodes, circuits), upward motion
Colors: Professional blue (#2563EB) with white and subtle purple accents
Mood: Confident, progressive, successful
Composition: Asymmetric design with clear space for text on left side

This will be used as a slide background with white text overlay.
```

### Exercise 2: Feature Announcement Visual

```
Create an image announcing a new feature: "Notification System MVP"

Subject: Communication and connectivity concept
Show: Notification icons, bell symbols, connected devices
Style: Modern product illustration
Colors: Bright, engaging, tech-forward
Mood: Exciting, innovative, helpful
Include: Mobile phone and desktop showing notifications

Make it suitable for a presentation slide or Teams announcement.
```

### Exercise 3: Metrics Visualization

```
Create a visual representation of team success metrics.

Concept: Upward trend, achievement, growth
Show: Abstract representation of metrics improving (not a literal chart)
Style: Clean illustration with geometric elements
Colors: Green for success, blue for baseline
Include: Arrows pointing up, celebration elements
Mood: Triumphant, data-driven, positive

This will accompany actual metrics in a presentation.
```

## Retrospective Facilitation Aids

Visual retrospective formats are more engaging than text-only.

### Exercise 4: Sailboat Retrospective Board

```
Create a visual board for a "Sailboat" retrospective format.

Layout: Side view scene with:
- A sailboat in the center
- Wind blowing from behind (represents what's helping us)
- An anchor below the boat (represents what's slowing us down)
- Rocks ahead in the water (represents risks/obstacles)
- An island in the distance (represents our goal)

Style: Clean, colorful illustration suitable for a Miro board
Colors: Blue water, green island, warm sun
Mood: Inviting, collaborative, metaphorical
Include: Labels or areas for each section

Make it suitable for a team to add virtual sticky notes to each area.
```

### Exercise 5: 4Ls Retrospective Visual

```
Create a visual for a "4Ls" retrospective format.

The 4Ls are:
- Loved (heart) - What we loved
- Learned (lightbulb) - What we learned
- Lacked (gap/missing) - What we lacked
- Longed For (star/wish) - What we longed for

Layout: Four quadrants or sections, each with its symbol
Style: Friendly, engaging illustration
Colors: Different color for each L
Mood: Reflective, positive, growth-oriented
Include: Clear visual symbols for each L

Make it suitable for screen sharing and sticky note placement.
```

### Exercise 6: Starfish Retrospective Visual

```
Create a visual board for a "Starfish" retrospective.

The 5 sections (star points):
- Keep Doing (center or top)
- Stop Doing
- Start Doing
- More Of
- Less Of

Layout: Starfish or star shape with 5 sections
Style: Clean, modern, collaborative
Colors: Different color for each point, harmonious palette
Mood: Action-oriented, balanced, constructive
Include: Clear labels for each section

Make it engaging for a team reflection session.
```

## Process Documentation Visuals

Visuals can make documentation more engaging.

### Exercise 10: Definition of Done Visual

```
Create an illustration representing "Definition of Done."

Concept: Checklist completion, quality gate
Show: Checkmarks, quality symbols, completion concept
Style: Clean, professional, infographic-style
Include:
- Checklist visual element
- Quality/seal of approval symbol
- Green "complete" color

Mood: Thorough, quality-focused, clear
This will accompany our written Definition of Done document.
```

### Exercise 11: Sprint Lifecycle Visual

```
Create an illustration of the sprint lifecycle.

Show the cycle:
Sprint Planning → Daily Standups → Sprint Review → Retrospective → (repeat)

Style: Circular or cyclical diagram illustration
Colors: Different color for each ceremony
Include: Simple icons representing each ceremony
Mood: Continuous, flowing, rhythmic

Make it suitable for onboarding new team members.
```

```
Create a set of 4 simple icons for Scrum ceremonies:

1. Sprint Planning icon - calendar/planning symbol

3. Sprint Review icon - presentation/demo symbol

Colors: Monochrome or consistent blue
Size: Should work as small icons (64x64 px concept)

These will be used consistently across team documentation.
```

## Quick Reference Card

### Essential Prompts

**Standup Analysis:**
```
Analyze standup notes. Extract blockers, patterns, actions for SM.
```

**Retro Synthesis:**
```
Synthesize into themes. Top 3 good, top 3 improve, 3 max actions.
```

**Ceremony Prep:**
```
Prepare facilitation guide for [ceremony]. Include agenda, questions, watch-fors.
```

**Coaching Prep:**
```
Prepare coaching conversation with [name]. Context, questions, avoid, outcomes.
```

**Escalation Draft:**
```
Draft escalation to [person]. Data, impact, tried, proposed, clear ask.
```

**Image Generation:**
```
Create [subject]. Style: [type]. Mood: [feeling]. Colors: [palette].
```
